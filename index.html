HTML

  • Gives the content structure
  • The head element contains information about the webpage.
  • The body element contains the visible content shown to the user.
  • TAG- Refers only to what appears inside the angle brackets.
  • ELEMENT- Refers to the entire seciton from opening to closing tag.

CSS

  • A margin indicates how much space we want outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: Check what branch we are currently on
  • git checkout -b branch-name: Creates a new branch and switches to it
  • git add -A: Commit all files in the working branch
  • git commit -m "description": To associate a message with the commit
  • git checkout main: Navigates you to the main branch
  • git pull origin main: Pulls the remote branch to the local branch and takes you to the most current/updated main branch
  • git push origin something: Pushes the changes made locally to the remote GitHub branch

JavaScript

  • var - A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer excutes code in a script. From top to bottom of page.
  • if- When using the if statement, code will only excute if the statement meets a condition, or is true.
  • for - performing a tast on each piece of groups of related data (array).
